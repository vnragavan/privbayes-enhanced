# Examples

Example scripts showing how to use Enhanced PrivBayes.

## Running Examples

### Basic example
```bash
python example.py
```
Shows basic usage with sample data.

### Generate all metrics
```bash
python generate_all_metrics.py
```
Runs all metrics: utility, privacy, QI linkage, inference, downstream, and DP audit.

### K-anonymity demo
```bash
python demo_k_anonymity.py
```
Shows what k-anonymity violations look like with real data.

## Files

- `example.py` - Basic usage
- `generate_all_metrics.py` - All metrics
- `demo_k_anonymity.py` - K-anonymity demo

## Usage

Run from the examples directory:
```bash
cd examples
python example.py
```

Or from project root:
```bash
python examples/example.py
```
